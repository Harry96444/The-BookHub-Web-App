🛑 Features 🛑


🎯 User Panel :

📍 User can buy books 
📍 User can sell old books
📍 User can manage orders 
📍 User will have its own cart

🎯 Admin Panel :

📍 only Admin has the authority of adding books
📍 Admins can manage orders of users
📍 Admins can see all books available
